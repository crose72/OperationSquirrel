Curriculum Overview

Turn-key robotics education built on real-world engineering tools.

What Students Learn

  • Linux, Docker, and Git fundamentals
  • GPU programming basics (CUDA)
  • Computer vision (OpenCV, YOLOv8)
  • Real-time data pipelines
  • Control systems: PID, Kalman filtering
  • MAVLink messaging & autonomous flight
  • Data logging and analysis using MCAP

Course Structure

  1. Introduction to Robotics & Safety
  2. Getting Started with Jetson Orin Nano
  3. Running Your First AI Detection Model
  4. Understanding Tracking & Localization
  5. Building Simple Autonomous Behaviors
  6. Flight Control Theory (PID, filters)
  7. Simulation → Real Drone Pipeline
  8. Capstone Project: Full Autonomous Demo

Sample Lessons

  • Lesson 1: Booting Jetson, SSH, Linux basics
  • Lesson 5: Real-time object detection
  • Lesson 7: MAVLink commands for control
  • Lesson 10: Autonomous AI-following mode
  • Capstone: Build and test a custom autonomy feature
